# Uloha

Uloha (Slovak for “task”) is a simple and lightweight task manager application. It helps users create, organize, and track their daily tasks in a clean and intuitive interface.

## Building
```console
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./appuloha
```

## Dependencies
* С++11 (or later)
* Qt 6.x
    * Core
    * Quick (QML)
    * SQL
* CMake 3.16+
